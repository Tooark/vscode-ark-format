# Manifest - Ark Format: PowerShell

Complete explanation of the `package.json` manifest file for the Ark Format: PowerShell extension.

## Package Metadata

```json
{\n  "name": "ark-format-powershell",\n  "displayName": "%displayName%",\n  "description": "%description%",\n  "version": "1.0.0",\n  "publisher": "tooark"\n}
```

### Fields

| Field | Value | Purpose |\n|-------|-------|---------|
| `name` | `ark-format-powershell` | Internal identifier (lowercase, no spaces) |\n| `displayName` | `%displayName%` | User-facing name (localized) |\n| `description` | `%description%` | Extension description (localized) |\n| `version` | `1.0.0` | Semantic version |\n| `publisher` | `tooark` | Publisher identifier |\n\n### Localization Keys

- **`%displayName%`** → `Ark Format: PowerShell`\n- **`%description%`** → Full description of the formatter\n\n## Engine & License\n\n```json\n{\n  \"engines\": {\n    \"vscode\": \"^1.85.0\"\n  },\n  \"license\": \"MIT\",\n  \"repository\": {\n    \"type\": \"git\",\n    \"url\": \"https://github.com/Tooark/ark-format-powershell\"\n  }\n}\n```\n\n### Fields\n\n| Field | Purpose |\n|-------|--------|\n| `engines.vscode` | Minimum VS Code version required (1.85.0+) |\n| `license` | License type (MIT) |\n| `repository` | Source code repository URL |\n\n## Categories & Keywords\n\n```json\n{\n  \"categories\": [\"Formatters\"],\n  \"keywords\": [\n    \"formatter\",\n    \"powershell\",\n    \"ps1\",\n    \"esbuild\",\n    \"pnpm\",\n    \"tooark\"\n  ]\n}\n```\n\n### Purpose\n\n- **`categories`**: Marketplace category (helps discoverability)\n- **`keywords`**: Search terms for the extension\n\n## Activation & Entry Point\n\n```json\n{\n  \"activationEvents\": [\n    \"onLanguage:powershell\"\n  ],\n  \"l10n\": \"./l10n\",\n  \"main\": \"./dist/extension.js\"\n}\n```\n\n### Fields\n\n| Field | Purpose |\n|-------|--------|\n| `activationEvents` | When to activate the extension |\n| `l10n` | Localization directory |\n| `main` | Entry point after bundling |\n\n### Activation Events\n\n- **`onLanguage:powershell`**: Activate when a PowerShell file is opened\n  - Ensures low startup overhead\n  - Only loads when needed\n\n## Contributes - Languages\n\n```json\n{\n  \"contributes\": {\n    \"languages\": [\n      {\n `.psd1` | PowerShell Module Manifest |\n| `.ps1xml` | PowerShell XML Configuration |\n\n### Language ID\n\nThe `id: \"powershell\"` matches VS Code's built-in PowerShell language support.\n\n## Contributes - Configuration\n\n```json\n{\n  \"configuration\": {\n    \"title\": \"%configuration.title%\",\n    \"properties\": {\n      \"arkFormatPowerShell.enabled\": { ... },\n      \"arkFormatPowerShell.indentSize\": { ... },\n      // ... more properties\n    }\n  }\n}\n```\n\n### Configuration Properties\n\nEach property defines a user-configurable setting:\n\n#### `arkFormatPowerShell.enabled`\n```json\n{\n  \"type\": \"boolean\",\n  \"default\": true,\n  \"description\": \"%config.enabled.description%\"\n}\n```\n- Enable/disable formatter\n- Type: Boolean\n- Default: Enabled\n\n#### `arkFormatPowerShell.indentSize`\n```json\n{\n  \"type\": \"number\",\n  \"default\": 4,\n  \"minimum\": 0,\n  \"description\": \"%config.indentSize.description%\"\n}\n```\n- Indentation level (in spaces)\n- Type: Number\n- Default: 4 spaces\n- Minimum: 0\n\n#### `arkFormatPowerShell.indentStyle`\n```json\n{\n  \"type\": \"string\",\n  \"default\": \"space\",\n  \"enum\": [\"space\", \"tab\"],\n  \"description\": \"%config.indentStyle.description%\"\n}\n```\n- Use spaces or tabs\n- Type: Enum\n- Options: `space`, `tab`\n- Default: `space`\n\n#### `arkFormatPowerShell.trimTrailingWhitespace`\n```json\n{\n  \"type\": \"boolean\",\n  \"default\": true,\n  \"description\": \"%config.trimTrailingWhitespace.description%\"\n}\n```\n- Remove trailing spaces\n- Type: Boolean\n- Default: Enabled\n\n#### `arkFormatPowerShell.maxConsecutiveBlankLines`\n```json\n{\n  \"type\": \"number\",\n  \"default\": 1,\n  \"minimum\": 0,\n  \"description\": \"%config.maxConsecutiveBlankLines.description%\"\n}\n```\n- Maximum blank lines between code\n- Type: Number\n- Default: 1 line\n- Minimum: 0\n\n#### `arkFormatPowerShell.removeLeadingBlankLines`\n```json\n{\n  \"type\": \"boolean\",\n  \"default\": true,\n  \"description\": \"%config.removeLeadingBlankLines.description%\"\n}\n```\n- Remove blank lines at start\n- Type: Boolean\n- Default: Enabled\n\n#### `arkFormatPowerShell.insertFinalNewline`\n```json\n{\n  \"type\": \"boolean\",\n  \"default\": true,\n  \"description\": \"%config.insertFinalNewline.description%\"\n}\n```\n- Add newline at EOF\n- Type: Boolean\n- Default: Enabled\n\n#### `arkFormatPowerShell.lineEnding`\n```json\n{\n  \"type\": \"string\",\n  \"default\": \"Auto\",\n  \"enum\": [\"LF\", \"CRLF\", \"Auto\"],\n  \"description\": \"%config.lineEnding.description%\"\n}\n```\n- Line ending style\n- Type: Enum\n- Options:\n  - `LF`: Unix/Linux (\\n)\n  - `CRLF`: Windows (\\r\\n)\n  - `Auto`: Auto-detect from file\n- Default: `Auto`\n\n#### `arkFormatPowerShell.effectLanguages`\n```json\n{\n  \"type\": \"array\",\n  \"items\": { \"type\": \"string\" },\n  \"default\": [\"powershell\"],\n  \"description\": \"%config.effectLanguages.description%\"\n}\n```\n- Languages to format\n- Type: Array of strings\n- Default: `[\"powershell\"]`\n\n#### `arkFormatPowerShell.useEditorConfig`\n```json\n{\n  \"type\": \"boolean\",\n  \"default\": false,\n  \"description\": \"%config.useEditorConfig.description%\"\n}\n```\n- Use .editorconfig settings\n- Type: Boolean\n- Default: Disabled\n\n## Contributes - Commands\n\n```json\n{\n  \"commands\": [\n    {\n      \"command\": \"arkFormatPowerShell.formatDocument\",\n
