# 🚩 Ark Format: Makefile - Start Here

Bem-vindo ao Ark Format: Makefile, um formatador seguro e configurável para Makefiles no VS Code.

Este arquivo organiza a leitura dos materiais de `samples/`.

## 📋 Ordem de leitura sugerida

1. [`01-QUICK-START.md`](https://github.com/Tooark/vscode-ark-format/blob/main/packages/makefile/samples/01-QUICK-START.md) - caminho mais curto para testar a extensão.
2. [`02-USER-GUIDE.md`](https://github.com/Tooark/vscode-ark-format/blob/main/packages/makefile/samples/02-USER-GUIDE.md) - guia completo de configuração e uso.
3. [`03-FEATURE-INDEX.md`](https://github.com/Tooark/vscode-ark-format/blob/main/packages/makefile/samples/03-FEATURE-INDEX.md) - referência por feature e por caso de uso.
4. [`04-TECHNICAL-SUMMARY.md`](https://github.com/Tooark/vscode-ark-format/blob/main/packages/makefile/samples/04-TECHNICAL-SUMMARY.md) - visão técnica detalhada dos exemplos.
5. [`05-STRUCTURE.md`](https://github.com/Tooark/vscode-ark-format/blob/main/packages/makefile/samples/05-STRUCTURE.md) - mapa visual da pasta e fluxos de navegação.
6. [`06-MANIFEST.md`](https://github.com/Tooark/vscode-ark-format/blob/main/packages/makefile/samples/06-MANIFEST.md) - inventário completo de arquivos e cobertura.
7. [`07-FINAL-REPORT.md`](https://github.com/Tooark/vscode-ark-format/blob/main/packages/makefile/samples/07-FINAL-REPORT.md) - relatório histórico/conclusivo do pacote.

## 🚀 Começar em 2 minutos

```bash
code samples/01-QUICK-START.md
code samples/example.mk
```

No VS Code, use `Shift + Alt + F` para formatar.
