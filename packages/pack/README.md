# Ark Format

Extension pack for the Ark Format family.

This package is the unified option in a 3-extension model:

- `ark-format-shell`: install only Shell formatting support.
- `ark-format-powershell`: install only PowerShell formatting support.
- `ark-format`: install both Shell and PowerShell formatters together.

## Included extensions

- Ark Format: Shell
- Ark Format: PowerShell
