# Release shell1.0.5

## Alterações desta versão

- Bump de versão do pacote `ark-format-shell`: `1.0.4` -> `1.0.5`.
- Reestruturação completa da documentação do pacote para o novo padrão de README:
  - adição de badges de versão/download (Open VSX) e licença.
  - reorganização das seções para fluxo padronizado (`Features`, `Getting Started`, `How to Use`, `Supported File Types`, `Configuration`, `Tips & Best Practices`).
  - ajustes de consistência em headings, exemplos e textos de uso.
- Inclusão de documentação em Português Brasileiro no pacote:
  - criado `README.pt-BR.md` com conteúdo equivalente ao README principal.
  - adicionada navegação de idiomas entre README em EN e PT-BR.
- Correções de consistência de configuração na documentação:
  - alinhado `lineEnding` para `LF` no exemplo principal, conforme default da extensão.
  - corrigida dica de uso para referenciar arquivos Shell (em vez de PowerShell).
- Criado arquivo de `CHANGELOG.md` para registrar mudanças no pacote shell.
