# Release pack1.0.0

Primeira versão do **Ark Format** — o extension pack que reúne todos os formatadores da família Ark Format em uma única instalação.

## O que está incluído

Este pacote instala automaticamente as seguintes extensões:

| Extensão                   | Descrição                                                                                                     |
| -------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **Ark Format: Shell**      | Formatação para scripts Shell (.sh, .bash, .zsh, .ksh, .tcsh, .azcli, .bats) com integração opcional ao shfmt |
| **Ark Format: PowerShell** | Formatação para scripts PowerShell (.ps1, .psm1, .psd1) com indentação inteligente de blocos param            |

## Recursos Combinados

- Formatação inteligente — documentos completos ou seleções parciais
- Configurações independentes por linguagem
- Interface multilíngue (EN e PT-BR)
- Integração com .editorconfig
- Integração opcional com shfmt para Shell
- Indentação de bloco `param()` no PowerShell
- Leve e rápido — sem dependências externas obrigatórias

## Quando usar este pacote

| Cenário                                       | Extensão recomendada         |
| --------------------------------------------- | ---------------------------- |
| Trabalha apenas com Shell                     | Ark Format: Shell            |
| Trabalha apenas com PowerShell                | Ark Format: PowerShell       |
| Trabalha com ambos ou quer cobertura completa | **Ark Format** (este pacote) |
